# E-commerce Funnel Analysis

## Objective
To identify friction points in the user journey and provide data-driven product recommendations.

## Business Insights
* **The Problem:** We observed a significant drop-off at the 'Checkout' stage.
* **The Insight:** Users are adding items to their cart but not finishing the purchase.
* **Recommendation:** Test a "Guest Checkout" option to reduce account-creation friction.
